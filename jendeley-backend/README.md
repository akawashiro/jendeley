# jendeley
JSON-based paper organaizing software

## Install
```
npm install @a_kawashiro/jendeley -g
```

## How to scan your PDFs
```
jendeley scan --papers_dir <YOUR PDFs DIR>
```


## Launch jendeley UI
```
jendeley launch --db <YOUR PDFs DIR>/jendeley_db.json
```
